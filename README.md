# Portfolio v4

Portfolio website built with **Next.js**, **Tailwind CSS**, **Lucide
React**, and **Framer Motion**.\
Includes a **dynamic article system** with per-slug content loading.

## Tech Stack

-   Next.js 16
-   Tailwind CSS
-   Lucide React
-   Framer Motion

## Features

-   Dynamic articles by slug
-   SEO metadata generation
-   Smooth animations
-   Responsive design

## Setup

``` bash
npm install
npm run dev
```
